# Quiz Kimetsu no Yaiba

Projeto simples de Programação para Web desenvolvido com **HTML**, **CSS** e **JavaScript**.

O sistema apresenta um quiz com 10 perguntas para descobrir com qual personagem de **Kimetsu no Yaiba** o usuário mais se identifica.

## Personagens

- Tanjiro Kamado
- Inosuke Hashibira
- Zenitsu Agatsuma

## Funcionalidades

- Tela inicial com apresentação do projeto
- Quiz com 10 perguntas
- Cada pergunta possui 3 alternativas
- Sistema de pontuação para os personagens
- Cálculo de compatibilidade em porcentagem
- Exibição do personagem final com imagem e descrição
- Botão para reiniciar o quiz

## Estrutura do projeto

- `index.html` → estrutura da página
- `estilo.css` → estilização da interface
- `script.js` → lógica do quiz
